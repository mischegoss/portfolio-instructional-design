The Writing Challenge

The core challenge was translating LookML, a domain-specific modeling language, into steps precise enough to execute in a live IDE for users at various levels.

About This Lab

This hands-on lab is part of the Google Cloud Data Analytics Certificate, a program offered through Coursera and Google Skills.

The lab asks learners to interact directly with the Looker IDE, the same environment they will use to apply the skills on the job. Working against a 90-minute timer with a scored 100-point assessment, learners complete key LookML tasks.
